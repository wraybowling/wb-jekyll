How it Happened

Between 2007 and 2010, the Jazz Mutant Lemur was a highly desirable piece of hardware & software that allowed a musician to create their own multi-touch MIDI control surface. Lemurs were very expensive, however, with a standard unit costing $1,999. I had grown tired of waiting for the price to come down, and the iPad hadn't been invented yet, so I decided to build one of my own with Processing, a web cam, a cardboard box, and a clear airvent cover.

Given my knowledge of object oriented programming, it didn't seem too hard to make a singular module that understood one multi-touch event, and converted that coordinate in to a 0â€“128 ranged number for use as a MIDI output.

Features

Lemur Clone features the following input types

  1. Buttons
    • Momentary
    • Toggle
  2. Sliders
    • Relative
    • Absolute
    • X-orientation
    • Y-orientation
    • Both X and Y (pad)
  3. Knobs
    • Relative
    • Absolute
  4. Monome-style Grid of arbitrary resolution 16x16 recommended maximum
