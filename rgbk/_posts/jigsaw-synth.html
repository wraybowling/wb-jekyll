Jigsaw Synth is a procedural tool for generating a physical jigsaw puzzle. Patterns can then be exported to a computerized cutter or a template to be cut by hand.

Thought Process

Pieces are laid out in a grid then randomness is added by drawing pieces with voronoi cells. Various other settings turn piece edges into interlocking tabs. Puzzle Synth let's the user blend between big, small, easy, hard, wiggly, woggley, lumpy, or whatever.

Jigsaw Synth was written in JavaScript, is open source, and was programmed in a minimalist style so that the code is easy to read. Its graphics are generated by building a Scalable Vector Graphic (SVG) in real time.
